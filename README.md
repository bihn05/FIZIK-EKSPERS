# FIZIK-EKSPERS
涉獵廣泛的物理實驗！

## 紹介
C++/SFML 實現的物理實驗可視化模擬。直觀展示物理原理。

## 試驗佇列

### 電磁
 - **[ELENS]** (eetm/elens/) - 模擬電子在電場中的聚焦過程。
### 材料
 - **[STRING]** (mtr/elens/) - 模擬非理想原子鏈的運動。

## 速始

### 依賴
 - GCC/G++ STD C++17
 - SFML 2.5

### 編運
```bash
# 倉庫克隆
git clone https://github.com/bihn05/FIZIK-EKSPERS.git
cd FIZIK-EKSPERS

# 編電子束透鏡
cd eetm/elens
make clean run
```
